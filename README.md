# news-manager
这是一个基于 vue+element ui+springboot+jpa框架的前后端分离项目，利用axios进行接口调用与传值

需安装：nodejs,vue
vue中要安装的插件：axios、element ui 

这个项目用的是vue3 安装好了后可以在cmd中输入vue ui直接创建项目，具体教程自己百度即可
